# Protein Corona:

蛋白冠是指纳米粒子进入生物体液（如血液、细胞外基质等）后，其表面吸附了一层生物分子（主要是蛋白质）的现象。具体来说，当纳米粒子与生物体液接触时，液体中的蛋白质会迅速在纳米粒子表面吸附，形成一层覆盖层。

# 输入输出

**Input:** 纳米粒的定性&定量特征

**Output:** 不同类型的蛋白丰度（蛋白冠组成）
